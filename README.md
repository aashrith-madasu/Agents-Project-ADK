## Setup Virtual Environment
```
#  Create env
python -m venv .venv

# Activate env
source .venv/bin/activate

# Install requirements
pip install -r requirements.txt
```

## To launch web UI
```
adk web
```

## To run the locally on CLI
```
python weather_bot/run.py
```
